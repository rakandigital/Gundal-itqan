GUNDAL ITQAN
Web App Bantuan Hafazan al-Quran (Versi Percuma)

======================================

Gundal Itqan ialah sebuah web app ringkas yang dibangunkan untuk membantu proses hafazan al-Quran
melalui kaedah pengulangan berperingkat (gundal), dengan fokus kepada disiplin, kesedaran,
dan istiqamah.

Aplikasi ini TIDAK memaparkan teks al-Quran dan bukan pengganti mushaf.
Pengguna disarankan menggunakan mushaf fizikal sepanjang sesi hafazan.

--------------------------------------
FUNGSI UTAMA
--------------------------------------
• Rekod pengulangan hafazan ayat demi ayat
• Kaedah:
  - 5 kali baca (buka mushaf)
  - 5 kali hafaz (tutup mushaf)
  - 10 set pengulangan bagi setiap ayat
• Fasa recap automatik sebelum berpindah ke ayat seterusnya
• Rekod progress hafazan boleh dimuat turun (CSV)
• Tidak memerlukan pendaftaran akaun
• Web app terus guna tanpa perlu install

--------------------------------------
PRINSIP REKA BENTUK
--------------------------------------
• Fokus kepada mushaf, bukan skrin
• Kurangkan distraksi visual
• Satu tugasan pada satu masa
• Sesuai digunakan dengan satu tangan (mobile-friendly)
• Versi percuma tanpa iklan

--------------------------------------
LIMITASI VERSI PERCUMA
--------------------------------------
• Data tidak disimpan secara kekal di server
• Rekod hafazan perlu dimuat turun secara manual
• Tiada sokongan multi-peranti / akaun
• Tiada paparan teks al-Quran

--------------------------------------
PERANCANGAN MASA DEPAN (OPSYENAL)
--------------------------------------

• Integrasi API al-Quran yang sahih
• Statistik hafazan jangka panjang
• Sokongan multi-surah & multi-sesi
